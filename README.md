# Jivanta Global Revenue OS CRM

Jivanta Global Revenue OS is a modern, enterprise-grade, real-time synced B2B CRM designed for commodity trading, domestic sales, and global exports.

## Features

- ☁️ **Real-time Firebase Sync**: Instant synchronization across devices with fully offline IndexedDB backups.
- 🔒 **Enterprise Authentication**: SHA-256 hashed login credentials, lockout protections, and session activity timeouts.
- 👥 **Role-Based Access Control (RBAC)**: Secure page guarding and custom permission overrides for managers and employees.
- 📊 **Dynamic Dashboard Filters**: Interactive sorting, "Date Added" tracking, and analytics filters by date range and lead owners.
- 🔄 **Action Loggers & unified timelines**: Interactive notes loggers, standardized call histories, and task calendars.

## Deploying to Vercel

This repository is pre-configured with `vercel.json` for SPA clean URL routing. 

1. Link this repository to your Vercel account.
2. Deploy.
3. Access Settings ➔ Database Operations in the CRM to sync and push your records.
